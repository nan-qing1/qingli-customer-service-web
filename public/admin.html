后端工作流

客服中台
1. 接入层与会话 ID统一承接视频、公众号、官网和 H5 入口，生成系统会话 ID / 客户 ID。
2. 知识库检索与客服回复只调用已审核资料生成对外回复，保证自超滑技术、空心杯电机和应用场景口径一致。
3. 资料采集卡仅在客户主动打开并提交后记录姓名、联系方式、希望交流内容和其他；再次打开自动回填并覆盖更新。
4. 客户档案主表同一客户只维护一份主档案，自动沉淀关注点、兴趣点、重复问题、询问深度和跟进建议。
5. 资料加入与版本审核后台录入、文件上传、批量导入或 API 资料先进入待审核区，审核通过后更新知识库版本。
6. 导入导出与后台分析导出客户档案、会话摘要和标签统计；统计高频问题、内容缺口和媒体内容优化方向。

前台合规

前台必须展示隐私提示和显式同意勾选框；未勾选不得提交资料采集卡。关闭采集卡不影响继续咨询。

敏感字段管理

手机号、微信、邮箱等字段默认隐藏或脱敏展示，仅授权人员可查看和导出。

内部权限

业务人员看客户档案和跟进建议，运营人员看分析和内容缺口，管理员维护资料、版本和权限。

审计与留存

资料更新、导入导出、人工修改都需要留痕；客户数据设置保存期限，超期删除或匿名化。
